# 🛡️ ResinGuard: 基于树莓派与国密 SM2 的 3D 打印机液面安全监测系统

这是一个面向 SLA 3D 打印工业场景的物联网安全网关与实时监控系统。本项目基于树莓派构建边缘计算网关，通过工业传感器采集树脂槽液面数据，使用国密 SM2 算法进行端到端加密传输，并在云端引入马氏距离算法，实现高精度的设备漏液与异常消耗隐性故障检测。

## ✨ 核心特性

* **边缘计算与协议转换**：基于 Raspberry Pi 4B，向下兼容 RS485/Modbus 工业传感器，向上转换为轻量级 MQTT 协议上云。
* **军工级数据安全**：采用中国国家密码标准 SM2 算法（非对称加密，C1C3C2 标准排列），确保生产设备运行数据在公网传输过程中的绝对防窃听与防篡改。
* **智能故障诊断 **：摒弃传统的绝对值上下限报警，采用马氏距离算法建立正常打印状态的协方差概率模型。精准过滤打印头上下运动引起的合理波动，敏锐捕捉漏液、传感器污染等隐性故障。
* **实时监控大屏**：提供基于 Matplotlib 的动态双屏仪表盘，同步展示物理液面距离与多维异常波动指数 (MD)。

---

## 🏗️ 系统架构

系统分为**边缘网关端 (Raspberry Pi)** 与 **云端控制台 (PC)** 两大部分。

1. **传感器层**：RS485 激光/超声波测距传感器实时获取树脂液面深度。
2. **边缘网关层(位于树莓派) (`main.py`)**：树莓派读取传感器数据 -> 组装 JSON -> SM2 公钥加密 -> 推送至 MQTT 代理。
3. **云端解析层 (`cloud_monitor.py`)**：订阅 MQTT 话题 -> SM2 私钥解密 -> 提取物理量 -> 计算马氏距离 -> 故障状态判定 -> 落盘持久化存储 (CSV)。
4. **前端展示层 (`realtime_plot.py`)**：高频读取日志，动态渲染工业级数据仪表盘。

---

## 🛠️ 硬件需求

* **边缘网关主控**：Raspberry Pi 4B (或其它 Linux 兼容开发板)
* **协议扩展模块**：双通道 RS485 扩展板 (如 DFR0824)
* **传感器**：支持 Modbus-RTU 协议的 RS485 激光测距传感器或超声波传感器
* **云端/分析端**：任意运行 Windows/Linux/macOS 的 PC

## 🚀 快速启动指南
步骤 1：生成你自己的专属国密密钥对
在 PC 端运行密钥生成工具，获取一对全新的 SM2 密钥。
将终端输出的 PK (公钥) 替换到树莓派 sm2_crypto.py 中，将 SK (私钥) 替换到 PC 端 cloud_monitor.py 中。

```Bash
python generate_key.py
```



步骤 2：验证加密通道闭环
运行诊断工具，确保密钥匹配且加解密逻辑（mode=1）在你的系统中运行正常：
```Bash
python test_sm2.py
```

步骤 3：启动 PC 云端大脑与监控大屏
打开两个独立的终端窗口，分别运行：
```Bash
# 启动后台监听、解密与马氏距离诊断引擎
python cloud_monitor.py
# 启动前端动态监控仪表盘
python realtime_plot.py
```

步骤 4：启动边缘网关
在树莓派端执行主程序，开始采集数据并加密上报：
```Bash
python main.py
```

## 📦 软件依赖与安装

请在树莓派与 PC 端均安装 Python 3.8+ 环境，并通过以下命令安装核心依赖包：

```bash
pip install gmssl paho-mqtt numpy pandas matplotlib
```
